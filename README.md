# 📊 Excel Retail & Sales Analysis Project

This project was completed as part of a **Data Technician Bootcamp**, focusing on developing practical data analysis skills using **Microsoft Excel**. The dataset used in this project represents retail and sales transactions, allowing for hands-on exploration of cleaning, analysing, and visualising business data.

***

## 🔍 Project Overview

The goal of this project was to transform raw retail data into meaningful insights. Using Excel features such as formulas, sorting, filtering, PivotTables, and charts, the project demonstrates core data analysis competencies relevant to business intelligence and operational reporting.

***

## 🧠 Key Skills Demonstrated

### ✅ **Excel Formulas & Functions**

A broad range of formulas were used to analyse trends, extract features from data, and summarise performance:

*   **Basic Aggregations**
    *   `SUM()`, `AVERAGE()`
*   **Conditional Calculations**
    *   `SUMIF()`, `AVERAGEIF()`
*   **Date & Time Functions**
    *   `DATE()`, `MONTH()`, `YEAR()`
*   **Lookup & Reference**
    *   `VLOOKUP()`
*   **Dynamic Array Functions**
    *   `UNIQUE()` for deduplication
*   **Logical & Switch-based Logic**
    *   `SWITCH()` for categorisation and segmentation

These functions enabled dynamic calculations such as identifying top-selling categories, extracting monthly trends, and grouping products.

***

### 🔎 **Data Cleaning & Organisation**

*   Filtering and sorting to isolate trends and anomalies
*   Removing duplicates using dynamic arrays and built-in features
*   Standardising date fields and ensuring correct data types
*   Using structured tables to enhance formula reliability

***

### 📈 **Data Analysis with PivotTables**

PivotTables were used to generate flexible summary views, including:

*   Sales by product category
*   Monthly revenue trends
*   Store-level performance
*   Customer segmentation

PivotCharts were created to visually communicate key patterns.

***

### 📊 **Data Visualisation**

Charts produced in the project include:

*   Column and bar charts for comparing sales performance
*   Line charts showing time-based trends
*   Pie and doughnut charts for product category share
*   Slicers for interactive filtering

These visuals help retail stakeholders understand revenue drivers and seasonal behaviour.

***

## 🧾 **Project Files**

*   **Retail\_Sales\_Analysis.xlsx** – main workbook containing raw data, cleaned tables, PivotTables, and charts
*   **Documentation Sheet** – contains explanations of formulas, logic, and workflow
*   **Dashboard Sheet** – interactive KPIs generated using PivotTables and charts

***

## 🚀 Summary

This project showcases the practical application of Excel in a retail analytics context — combining formulas, structured data processing, visualisation, and dashboard creation to support data-driven decision‑making.

If you're exploring Excel for data analysis, this project demonstrates the essential toolkit for turning raw data into actionable insights.
